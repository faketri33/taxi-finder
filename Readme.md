
# Taxi finder

Общее описание проекта

Taxi Finder — это модульный backend‑проект, предназначенный для поиска, подбора и управления поездками такси. Проект построен по принципам чистой архитектуры, модульного монолита / микросервисной готовности и ориентирован на дальнейшее масштабирование.

### Основные цели проекта:
- Чёткое разделение доменной логики, инфраструктуры и общих компонентов
- Переиспользуемость кода между модулями
- Готовность к контейнеризации и CI/CD
- Удобство локальной разработки и деплоя
